# cs109-distributions
The website for the CS 109 Final Project Part 3. Calculates the expected value,
variance, and PMF/PDF of Random Variables.

## Distributions Supported
* Bernoulli
* Binomial
* Poisson
* Geometric
* Hypergeometric
* Negative Binomial
* Uniform
* Normal
* Exponential
